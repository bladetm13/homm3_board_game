# HOMM 3 Board Game | Режими гри

## Позначки

- 👪 - режим для певної к-ті гравців
- 🏆 - умови перемоги
- ⏰ - дії по закінченню раундів
- 🤕 - дії при вибуванні одного з гравців
- 🔹 - додаткові правила
- ❓ - орпціональне правило
- ➡️ - посилання на інший режим


## Об'єкти карти

### Обеліски
- 🔹 Якщо в сценарії не використовуються обеліски
- 🏆 нагорода за відвідування `пошук (6) у колоді артефактів`, `2 х Кубики скарбів`

### Утопя драконів
- 🔹 Якщо в сценарії не задіяна Утопія драконів - вона стає Банком створінь
- 🔹 Поле бою у Банку створінь (`E` - створіння-охоронець, `U` - місце для вашого юніта (не більше 5 ваших юнітів на полі бою)):
    ```
    [E][ ][ ][ ][E]
    [ ][U][U][U][ ]
    [ ][U][U][U][ ]
    [E][ ][ ][ ][E]
    ```
- 🔹 Охоронці: `1 x Чорний дракон`, `1 x Золотий дракон`, `1 x Кристальний дракон`, `1 x Фея-дракон`.
- 🔹 Відсутній ліміт раундів
- 🏆 Нагорода: `20 монет` і `Пошук (4) у колоді артефактів` двічі, випадкове `Лазурне нейтральне створіння`

### Грааль
- 🔹 Якщо в сценарії не задіяний грааль, після його отримання і доставки у місто
- АБО 🏆 Нагорода: добрати карти характеристик відповідно до картки героя (так, знову), всі карти характеристик завжди граються на експертному рівні
- АБО 🏆 Нагорода: мінімальна потужність усіх заклинань головного героя збільшується на +1
- АБО 🏆 Нагорода: для всіх карт артефактів спрацьовують обидва еффекти одночасно, правила які примушують "видалити" карту з гри, не видаляють її
- АБО 🏆 Нагорода: усі карти навичок героя завжди граються на експертному рівні
- АБО 🏆 Нагорода: ліміт карт у руці = 10 до кінця гри
